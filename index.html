Integração com Tribunais — Novas ações por CNPJTRT/PJe · captura de distribuições
Objetivo do módulo

Ao inserir o CNPJ do cliente, o Lawgico consulta os Tribunais do Trabalho/TRTs integrados para identificar novas ações distribuídas, validar se o processo já existe na base e enviar novos casos para a fila de cadastro.

Fluxo esperado

CNPJ → consulta TRT/PJe → novas distribuições → validação de duplicidade → enriquecimento dos dados → envio para fila de processos → criação de preparo, documentos e prazos.

Resultado da consultaAguardando CNPJ
Sel. Status TRT Data distribuição CNJ Cliente/CNPJ Parte contrária Vara Fase Ação
Informe um CNPJ e consulte para visualizar novas ações distribuídas.
Regras para o TIIntegração e segurança
1. Consulta por CNPJ

Permitir cadastro de CNPJs monitorados por cliente e consulta periódica aos TRTs/PJe disponíveis.

Monitoramento
2. Evitar duplicidade

Antes de importar, cruzar CNJ, partes, CNPJ e cliente com a base interna Lawgico.

Validação
3. Criar fila automática

Novas ações entram na fila de processos com status “Aguardando cadastro” e origem “TRT/PJe”.

Fila
4. Log de consulta

Registrar data, usuário, TRT consultado, retorno, erros e indisponibilidades.

Auditoria
5. Alertas

Quando novas ações forem encontradas, gerar alerta para Controladoria e responsável pelo cliente.

Notificação
6. Respeitar acesso

Cada usuário visualiza apenas clientes/CNPJs autorizados conforme perfil de acesso.
